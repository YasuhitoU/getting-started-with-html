How the web works

When a person goes to a web page, browser is sending http request through the internet to the servers which in return sends html document to the browser.

HTML

HTML stands for hyper text markup language. HTML documents contain text content whose appearance is controlled by markup.

Element and Tags

HTML documents are made of HTML elements, which are either block or inline. We define the type of elements by using tags.

Examples

"b" bold - inline element

"em" emphasis - inline element

"br"
break - inline element

"p" paragraph - block element
